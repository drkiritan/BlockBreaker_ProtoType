ゲームジャンル：ブロック崩し

遊び方：
タイトル画面からスペースを押すといきなりゲームが開始します。
ブロックをすべて消せばゲームクリアです。

→キー    カーソルが右に移動
←キー    カーソルが左に移動
spaceキー ボール発射

判定はちょっと甘めです（笑）。

tips：
残機は2つあり、ボールが落ちても一度は復活できますが、
リスタートするボールの挙動がわからず即2球目も落下してしまう
ケースが予想されます（修正？？なんのことやら）。

ボールは最初必ず右斜め下に向かって進むので、カーソルをそこに
合わせてからスペースキーを押して再開してください。
